Sparta

Build & deploy Go functions in AWS Lambda

"No Server Is Easier To Manage Than No Server."

Werner Vogels AWS re:Invent 2015

Sparta provides a framework to build & deploy Go functions in AWS Lambda. While Go is not yet officially supported by AWS Lambda (see poll), it’s possible to bundle & launch arbitrary executables in Lambda.

Sparta provides a HTTP-based translation layer between the proper NodeJS environment and your Go binary. In addition to this translation layer, Sparta is also able to:

  • Manage S3 and SNS-based event sources.
    • A Lambda function may be invoked in response to S3 or SNS broadcasted events.
  • Provision an HTTPS API Gateway service that allows Lambda functions to be publicly invoked.
  • Produce diagrams of Lambda & event source interactions
